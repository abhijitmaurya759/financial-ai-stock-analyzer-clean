# 📊 AI Financial Stock Analyzer

An AI-powered web app that analyzes stock data, news sentiment, and generates professional PDF reports.

## 🚀 Features

* 📈 Stock price visualization (Indian & Global stocks)
* 📰 Latest news fetching
* 🤖 AI sentiment analysis using OpenAI
* 📄 Downloadable PDF report
* ⚡ Fast performance with caching
* 🌐 User-friendly Streamlit interface

## 🛠️ Tech Stack

* Python
* Streamlit
* OpenAI API
* Yahoo Finance (yfinance)
* NewsAPI
* Matplotlib
* FPDF

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔐 Setup API Keys

Create a `.env` file:

```
OPENAI_API_KEY=your_key
NEWS_API_KEY=your_key
```
